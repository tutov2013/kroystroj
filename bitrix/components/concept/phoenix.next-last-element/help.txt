Компонент добавляет на страницу ссылки вида "Сделующий товар" и "Предыдущий товар". Можно использовать для новостей, статей и тд. Компонент выводит ссылки на товар в той же подкатегории, что и он сам. Если товар первый или последний в категории выводится,соответственно, первый или последний товар другой подкатегории. Надеюсь принцип понятен.

Пример вызова компонента:

<?
$APPLICATION->IncludeComponent("bitrix:next_last_element","",Array(
		"ELEMENT_ID" =>  $arResult["ID"],
		"NEXT_NAME" => "Следующий товар",
		"LAST_NAME" => "Предыдущий товар",
		"CACHE_TYPE" => "Y",
		"CACHE_TIME" => "3600000",
		"CACHE_GROUPS" => "Y"
	)
);
?>

Описание ключей:

ELEMENT_ID -- ID текущего элемента
NEXT_NAME  -- Имя кнопки для перехода к следующему элементу
LAST_NAME  -- Имя кнопки для перехода к предыдущему элементу
